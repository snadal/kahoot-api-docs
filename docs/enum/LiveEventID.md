## LiveEventID
A special identifier for various Kahoot Live events.

- `1` - GetReady

- `2` - QuestionStart

- `3` - GameEnd

- `4` - TimeUp

- `5` - GameRestart

- `6` - ~SelectAnswer~

- `7` - ~AnswerResponse~

- `8` - QuestionEnd

- `9` - QuizStart

- `10` - QuizEnd

- `11` - SendFeedback

- `12` - RequestFeedback

- `13` - RevealRank

- `14` - AcceptName

- `15` - ~DenyName~

- `16` - RequestRecoveryData

- `17` - SendRecoveryData

- `18` - JoinTeam

- `19` - ReceiveTeam

- `20` - TeamTalkStart

- `21` - ~TeamTalkEnd~

- `30` - ~PauseGame~

- `31` - ~IFrameEvent~

- `32` - ~ServerIFrameEvent~

- `40` - ~StoryBlockGetReady~

- `41` - ~ReactionSelect~

- `42` - ~ReactionResponse~

- `43` - ~GameBlockStart~

- `44` - ~GameBlockEnd~

- `45` - AnswerQuestion

- `50` - SubmitTwoFactor

- `51` - InvalidTwoFactor

- `52` - ValidTwoFactor

- `53` - TwoFactorReset
